# Aura TodoList

## Project Overview
Aura TodoList is a simple web-based task management application that allows users to add and remove daily tasks easily.

## Role: Full-Stack Developer
The Full-Stack Developer was responsible for designing and implementing the user interface and client-side functionality of the application.

## Features Implemented
- Add new tasks
- Remove tasks by clicking on them
- Clean and simple user interface
- Responsive layout using basic CSS

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
